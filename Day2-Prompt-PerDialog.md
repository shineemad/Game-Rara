# Prompt ChatGPT PER-DIALOG (DETAIL) — HARI 2 · RARA: Jaga Dirimu!

> Versi **detail-tinggi**: tiap prompt sudah berisi **shot/komposisi, posisi & ekspresi
> karakter, aksi spesifik sesuai dialog, environment, pencahayaan/waktu, palet warna,
> dan mood**, sehingga hasil gambar persis menggambarkan situasi dialognya.
>
> **Cara pakai:** lampirkan Gambar 1 (referensi) → untuk tiap dialog tempel
> `[STYLE] + [CAST] + [SCENE]` → generate. Semua adegan **non-grafis & child-safe**.
>
> ⚠️ **Kunci konsistensi visual:** SELALU tempel **4 blok wajib** ini di awal SETIAP
> prompt dengan urutan tetap → `STYLE BLOCK` → `COLOR LOCK` → `CHARACTER LOCK (CAST)`
> → `ENVIRONMENT BLOCK`, baru `SCENE`. Lampirkan **Gambar 1 + character sheet Rara**
> sebagai image reference tiap generate, dan pakai **seed yang sama**.

## 🎨 STYLE BLOCK (wajib tempel di setiap prompt)

```
Studio Ghibli / Isao Takahata-inspired hand-painted 2D anime still. Soft
watercolor & gouache textures, visible paper grain, painterly cel-shading with
delicate gradients and soft rim light. Naturalistic Indonesian kampung/urban
setting, lush detailed backgrounds, atmospheric depth, gentle bokeh on distant
elements. Warm earthy palette: cream, terracotta, teak-brown, sage green, dusty
lavender, deep maroon. Wholesome, tender, child-friendly tone. Clean composition,
no text, no UI, no logos, no watermark. 16:9 cinematic aspect. Match the art
style, line quality, character design and color palette of the attached reference
image.
ENVIRONMENT QUALITY: every location must look CLEAN, TIDY and WELL-MAINTAINED —
swept streets, neatly trimmed plants, freshly painted walls, polished floors, no
litter, no trash, no graffiti, no rust, no broken or grimy surfaces. Even the
"dim/quiet" spots stay orderly and cared-for, never dirty or run-down.
```

## 🧹 ENVIRONMENT BLOCK (wajib tempel — lingkungan bersih & terawat)

## 📍 SET CONTINUITY (lokasi berulang harus tampak SAMA tiap muncul)

```
Recurring sets must look identical every time they reappear:
• ANGKOT INTERIOR: same teal-green vinyl bench seats, same cream ceiling, same
  window frames, same hanging trinket and route stickers, same driver seat — only
  the camera angle, passengers and lighting change.
• RARA'S BEDROOM: same bed with the same blanket pattern, same small wooden lamp,
  same window position.

Keep architecture, furniture layout and key props consistent; vary only framing.
```

## 👧 CHARACTER LOCK / CAST SHEET (wajib tempel di setiap prompt)

```
RARA (hero, 10–12 y.o. Indonesian girl): glossy black chin-length bob with blunt
straight fringe, soft round dark eyes, small nose, fair-tan skin. OUTFIT: purple
ribbed knit turtleneck sweater, long pleated dark-maroon skirt to mid-calf, white
ankle socks, black mary-jane shoes, tan/brown leather school backpack. Petite,
gentle yet brave.
STRANGER MAN (antagonist): ordinary adult Indonesian man, 30s–40s, plain
shirt/jacket, dark cap, faint stubble. Drawn NON-graphic & NON-scary — convey
danger ONLY through "too-friendly / invasive / uneasy" body language and Rara's
reaction. NEVER violent, sexual, frightening, or physically explicit.
PAK SUPIR: kind middle-aged angkot driver, short sleeve batik/plaid shirt, songkok
optional, reassuring face.
IBU (home only): warm mother in dusty-lavender floral hijab + matching floral
dress (as in reference image).
CONSISTENCY: keep Rara's exact face, hairstyle and outfit identical in every image.
RARA LOCK: same face proportions (round soft cheeks, large dark eyes, short straight
bob with blunt fringe), same height ratio (head ≈ 1/6 of body), same outfit colors
and details in EVERY shot — do NOT redesign, age up, or change clothing/hair.
FACE ANGLE: keep features readable and on-model from any angle; avoid distorted or
off-model faces.
```

## ✅ ATURAN KONSISTENSI & KEAMANAN

- **Selalu lampirkan Gambar 1** + frasa `match the attached reference image's style,
character design and palette`.
- Buat **character sheet Rara** sekali (depan/samping/3-4 ekspresi), pakai sebagai
  acuan wajah untuk SEMUA adegan.
- Pertahankan **outfit & wajah Rara identik** di tiap gambar; sebut ulang di tiap prompt.
- **16:9**, eye-level kecuali disebut low/high angle.
- Semua adegan pelaku **non-grafis, non-seksual, non-kekerasan, child-safe** — bahaya
  hanya lewat bahasa tubuh & reaksi Rara.
- Untuk adegan "akibat" (BAHAYA): tekankan **mood waspada/edukasi**, bukan ketakutan
  ekstrem.
- **Lingkungan selalu bersih & terawat** di setiap adegan (tanpa sampah, coretan,
  karat, cat mengelupas, atau kekotoran) — kesan kumuh datang hanya dari pencahayaan
  redup, bukan dari kotoran.

## 🔗 WORKFLOW KONSISTENSI VISUAL (urutan kerja yang disarankan)

1. **Generate character sheet Rara dulu** (tampak depan + samping + 3–4 ekspresi,
   latar polos). Pilih satu hasil terbaik = **acuan master**.
2. Tiap adegan: lampirkan **Gambar 1 + character sheet master** sebagai image
   reference, lalu tempel `STYLE + COLOR LOCK + CHARACTER LOCK + ENVIRONMENT +
SET CONTINUITY + SCENE`.
3. **Kunci seed** yang sama (atau fitur "same character/style") bila tool mendukung,
   supaya wajah & palet tidak berubah antar gambar.
4. Generate **set berulang** (angkot, kamar, ruang tamu) berurutan dalam satu sesi
   agar look-nya nyambung; simpan 1 hasil tiap set sebagai acuan untuk adegan lain.
5. Jaga **aspect ratio 16:9** dan jarak kamera yang konsisten untuk adegan sejenis.
6. Kalau wajah/warna mulai "melenceng", regenerate sambil menegaskan ulang
   `CHARACTER LOCK` + `COLOR LOCK` dan lampirkan kembali character sheet master.
